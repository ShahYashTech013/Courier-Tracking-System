Courier Tracking System

The Courier Tracking System is a simple web-based application designed to manage and track courier shipments efficiently. It allows users to track parcels using a unique tracking ID and view the current delivery status in real time.

The system also includes an admin panel that helps in managing shipment details and updating delivery progress, making the overall process organized and easy to handle.

🚀 Features
User registration and login
Shipment booking with sender & receiver details
Unique tracking ID generation
Track shipment status (Dispatched, In Transit, Delivered)
Admin panel to manage shipments and update status
👥 User Roles
User: Track shipments
Admin: Manage shipments
Delivery Staff: Update delivery status
🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: C# with ASP.NET
Database: SQL Server Express
⚙️ Requirements
Web browser (Chrome, Edge, etc.)
Internet connection
